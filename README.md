Salle de Sport

Ce projet est une application web permettant la gestion complète d’une salle de sport :  
cours, équipements, statistiques, associations, et l'export.

Le système offre un **dashboard dynamique**, un **CRUD complet**, un **système d’association cours/équipements**,  
ainsi qu’un **module d'export CSV**.


## Acteurs du système

### Administrateur
- Gère les cours
- Gère les équipements
- Gère les associations cours-équipements
- Accède au dashboard et statistiques
- Exporte les données en CSV

### Utilisateur
- Consulte les cours
- Consulte les équipements

# Fonctionnalités principales

##  Gestion des cours
- Ajouter / Modifier / Supprimer un cours  
- Voir la liste des cours  
- Afficher les cours par catégorie  
- Export CSV

## Gestion des équipements
- Ajouter / Modifier / Supprimer un équipement  
- Voir la liste des équipements 
- Afficher les équipements par type  
- Export CSV

## Dashboard + Statistiques
- Nombre total de cours  
- Nombre total d’équipements  
- Graphiques  
  - Répartition des cours par catégorie  
  - Répartition des équipements par type
